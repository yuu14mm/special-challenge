### URLとはなにか
"URL"とは「Uniform Resource Locator」の略称で、インターネット上に存在する文書や画像などの場所を指し示す技術方式.
```
http://ドメイン名/ディレクトリパス名/ファイル名
```
```
https://ドメイン名/ディレクトリパス名/ファイル名
```
という形式で構成されるホームページページのアドレス。
